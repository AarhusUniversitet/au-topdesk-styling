Shared UI improvements for TopDesk at Aarhus University, distributed via the Stylus browser extension.

Installation

  1. Install the Stylus browser extension
  2. Click the install button below
  3. Click "Install" in Stylus
Install AU TopDesk Styling
The style applies only to https://au.topdesk.net and updates automatically when changes are published.

Source code: github.com/AarhusUniversitet/au-topdesk-styling
