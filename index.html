AU TOPdesk Styling

Shared UI improvements for TOPdesk at Aarhus University. The styles are installed via the Stylus browser extension and update automatically.

Step 1: Install Stylus

Stylus is a browser extension that allows you to install custom styles.

  • Install Stylus for Chrome
  • Install Stylus for Firefox

Step 2: Open AU TOPdesk Styling in Stylus

Click the button below to open the style in Stylus.

Open AU TOPdesk Styling

Step 3: Install in Stylus

Click Install in Stylus

The style applies only to https://au.topdesk.net. It contains only visual/UI adjustments and updates automatically when changes are published.

Source code: github.com/AarhusUniversitet/au-topdesk-styling
